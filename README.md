# metronomelutter

操起吉他熟练地 53231323,然后发现市面上竟然没有一个干净无广告的节拍器?好吧那只能自己写一个了.

基于 flutter 技术打造的极简全平台(包括 web)节拍器,目前只是初版,后续把功能完善.
